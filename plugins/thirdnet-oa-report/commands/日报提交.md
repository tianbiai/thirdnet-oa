description: 提交 OA 日报（支持 AI 智能总结）
argument-hint: "[日期] - 例如: '今天' '2026-04-26' 或留空默认今天"
allowed-tools:
  - AskUserQuestion
---

提交 OA 日报。用户输入：$ARGUMENTS

按照 `thirdnet-oa-report/skills/submit-daily-report/SKILL.md` 中定义的 OA 日报提交 Skill 工作流程执行。

关键步骤：
1. 调用 MCP 工具 `check_config` 检查配置，未配置则引导 `/配置OA`
2. 调用 MCP 工具 `get_report_template` 获取模板字段
3. 收集日报内容（手动或 AI 智能总结）
4. **必须使用 AskUserQuestion 展示完整预览并等待用户确认，禁止自动提交**
5. 确认后调用 MCP 工具 `submit_report` 提交
