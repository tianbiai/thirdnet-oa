查看 OA 日报提交情况。用户输入：$ARGUMENTS

按照 `skills/view-daily-report/SKILL.md` 中定义的 OA 日报查看 Skill 工作流程执行。

关键步骤：
1. 调用 MCP 工具 `check_config` 检查配置，未配置则引导 `/配置OA`
2. 解析查询条件（日期、人员、部门）
3. 调用 MCP 工具 `get_reports` 和 `get_institutions` 查询数据
4. 格式化展示结果
